Usage Notes

  • Use btn-active class and aria-pressed="true" for toggle states
  • Button groups work with all button variants (primary, secondary, outline, ghost, etc.)
  • Split buttons are perfect for dropdown menus attached to primary actions
  • Vertical groups are great for sidebars and narrow layouts
  • All groups maintain proper z-index layering for hover and focus states
