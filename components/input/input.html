Accessibility Best Practices

  • Always use <label> elements with for attribute
  • Use appropriate input type attributes (email, tel, number, etc.)
  • Include placeholder text for guidance, but don't rely on it alone
  • Add required attribute for mandatory fields
  • Provide error messages with form-error class
  • Use form-description for helpful hints
  • Ensure sufficient color contrast for text and borders
  • Test with keyboard navigation (Tab, Shift+Tab, Enter)
